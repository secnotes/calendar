# 万年历

一个功能丰富的中文万年历网页应用，支持公历与农历显示、黄历查询、节假日提醒等功能。

## 功能特性

### 日历核心功能
- 公历日历显示，支持月份切换
- 农历日期显示（包括闰月）
- 二十四节气标注
- 传统节日显示（农历节日、公历节日）
- 今日高亮标记
- 周末日期红色标注
- 法定节假日标记（休/班）

### 黄历信息
- 宜忌事项
- 干支纪年（年柱、月柱、日柱、时柱）
- 纳音五行
- 冲煞方位
- 值神
- 建除十二神
- 星宿
- 月相
- 物候
- 吉神宜趋
- 凶神宜忌
- 彭祖百忌
- 胎神方位

### Widget组件
- 日期卡片：显示日期、年中天数、周数
- 老黄历卡片：四柱干支、宜忌事项
- 月历小卡片：迷你月历视图
- 节假日倒计时：显示最近的节假日

## 技术栈

- HTML5
- CSS3（响应式设计）
- JavaScript（原生）
- lunar.js（农历计算库）

## 使用方法

直接在浏览器中打开 `index.html` 文件即可使用：

```bash
# 方式一：直接打开
# 双击 index.html 文件

# 方式二：使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 响应式设计

应用支持多种屏幕尺寸：
- 桌面端（>768px）：双列布局
- 移动端（<768px）：单列布局，自适应卡片大小

## 依赖说明

本项目使用 lunar.js 库进行农历计算，该库提供了：
- 公历与农历日期转换
- 干支计算
- 节气计算
- 节日查询
- 黄历宜忌数据
